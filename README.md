<h2>Interactive Karate Page</h2>
Hi there! Welcome to this project.
This is a responsive, multi-layout educational webpage dedicated to the history, philosophy, and practices of traditional Karate. I built this from scratch using HTML5 and vanilla CSS to practice building more complex page architectures.

The challenge I gave myself was moving past basic, single-column blocks and trying to figure out how to build a real multi-zone layout that actually looks like a proper webpage. It features a hero banner header, a custom navigation bar, a multi-column flexbox main body, and an interactive sidebar gallery that utilizes CSS pseudo-classes (:hover) to control opacity and caption visibility.

<h2>Tech & Design</h2>
Flexbox Layout Engine: Used CSS Flexbox (display: flex) to construct a reliable, side-by-side main content area and sidebar, ensuring elements distribute evenly within an 1100px wrapper.

<p></p>
Interactive UI Effects:
Programmed pure CSS hover states (.img-container:hover) to dynamically toggle item visibility (visibility: visible) and smoothly adjust opacity settings without relying on JavaScript.

<p></p>
Internal Anchor Navigation: Implemented clean semantic ID tagging to create seamless on-page scrolling from the navigation bar directly down to specific structural sections.
